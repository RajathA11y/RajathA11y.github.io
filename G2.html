Guideline 1.2 Time-based Media: Provide alternatives for time-based media

Purpose

To provide access to time-based and synchronized media.

Benefits of this guideline

Allows access to all of the information in the synchronized media. This option also allows access to the visual information in non-visual form when audio description is not provided

Test procedure Expected result:
* Navigate to the media using screen readers (JAWS, NVDA, VoiceOver, and talkback) The link to the media must be meaningful; Verify that a link to a text transcript is provided immediately after the link to the media
* If captions are present, navigate to the captions Verify that the Captions provide the part of the content available via the audio track. Captions not only include dialogue, but identify who is speaking and include non-speech information conveyed through sound.

Reasons for Failure

  • Using text alternatives like filenames or placeholder text)
  • Providing long descriptions for non-text content that does not serve the same purpose or does not give the same information.
